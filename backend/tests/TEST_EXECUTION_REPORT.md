## 🐛 发现的问题

### 高优先级问题

#### 问题1: 购物车SQL查询错误
**位置**: `backend/src/controllers/cart.controller.js`
**错误**: `Unknown column 'sku->spu.name' in 'field list'`
**影响**: 购物车功能完全不可用
**优先级**: 🔴 高
**修复建议**:
```javascript
// 修改前（错误）
include: [{
  model: ProductSkus,
  as: 'sku',
  attributes: ['sku_name', 'price'],
  include: [{
    model: ProductSpus,
    as: 'spu',
    attributes: ['title', 'primary_image']
  }]
}]

// 修改后（正确）
include: [{
  model: ProductSkus,
  as: 'sku',
  attributes: ['sku_name', 'price', 'spu_id'],
  include: [{
    model: ProductSpus,
    as: 'spu',
    attributes: ['id', 'title', 'primary_image']
  }]
}]
```

### 中优先级问题

#### 问题2: 测试脚本响应格式不匹配

**影响**: 自动化测试通过率低，但API实际正常
**优先级**: 🟡 中
**修复建议**: 更新测试脚本以匹配实际API响应格式

#### 问题3: 部分API返回400错误

**优先级**: 🟡 中
