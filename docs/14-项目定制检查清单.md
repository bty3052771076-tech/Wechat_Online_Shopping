## 3. 商品数据配置

### 3.1 商品基本数据

**文件**: `model/good.js`

- [ ] **已了解价格单位**
  - ⚠️ **重要**：价格单位是"分"，不是"元"
  - 9900 = 99元
  - 29800 = 298元

- [ ] **商品 ID 唯一性**
  - [ ] 每个商品有唯一的 `spuId`
  - [ ] 每个SKU有唯一的 `skuId`

- [ ] **商品基本信息已填写**
  ```javascript
  {
    spuId: '1',                    // ✅ 唯一ID
    title: '商品名称',              // ✅ 商品标题
    primaryImage: '图片URL',       // ✅ 主图
    minSalePrice: 9900,           // ✅ 价格（分）
    spuStockQuantity: 100,        // ✅ 库存
    soldNum: 0                    // ✅ 已售数量
  }
  ```

- [ ] **商品图片已准备**
  - [ ] 本地图片：放在 `assets/` 目录
  - [ ] 网络图片：使用 HTTPS URL
  - [ ] 图片尺寸：建议 800x800 像素
  - [ ] 图片格式：JPG 或 PNG
  - [ ] 图片大小：单张 < 2MB

- [ ] **商品规格已配置**（如需要）
  ```javascript
  specList: [
    {
      specId: '1',
      specName: '颜色',
      specValues: [
        { specValue: '白色', specValueId: '1' },
        { specValue: '黑色', specValueId: '2' }
      ]
    }
  ]
  ```

- [ ] **SKU 已配置**（如需要）
  ```javascript
  skuList: [
    {
      skuId: '1',
      spuId: '1',
      skuName: '白色 S',
      price: 9900,
      stockQuantity: 50,
      specs: [
        { specId: '1', specValueId: '1' },  // 颜色：白色
        { specId: '2', specValueId: '3' }   // 尺码：S
      ]
    }
  ]
  ```
