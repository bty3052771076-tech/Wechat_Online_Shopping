## 📅 接下来4周的行动计划

### 第1周：理解项目 + 配置环境

#### 目标
- 理解示例项目结构
- 搭建后端开发环境
- 准备数据库

#### 具体任务

**任务1：理解项目结构（1天）**

打开开发者工具，查看文件列表，了解：

```
example/
├── pages/           # 页面（重点！）
│   ├── home/        # 首页
│   ├── category/    # 分类页
│   ├── cart/        # 购物车
│   ├── goods/       # 商品相关
│   ├── order/       # 订单相关
│   └── usercenter/  # 个人中心
├── services/        # 接口服务（重点！）
├── components/      # 组件
├── utils/           # 工具函数
├── app.js           # 小程序入口
├── app.json         # 全局配置
└── project.config.json  # 项目配置
```

**行动：**
1. 逐个点击 `pages/` 下的文件夹
2. 查看每个文件夹里的4个文件：`.js` `.json` `.wxml` `.wxss`
3. 理解：这4个文件构成一个完整的页面

**任务2：配置后端开发环境（2天）**

你需要后端服务来提供数据。

**选择方案：**

**方案A：使用Node.js（推荐）**
```bash
# 1. 确认已安装 Node.js
node --version

# 2. 在项目根目录创建后端文件夹
mkdir Wechat_Online_Shopping-backend
cd Wechat_Online_Shopping-backend

# 3. 初始化项目
npm init -y

# 4. 安装依赖
npm install express mysql2 redis cors body-parser
```

**方案B：使用微信云开发（最简单）**
- 小程序内置的云服务
- 不需要自己搭建服务器
- 适合快速开发

**我的建议：先使用方案B（云开发）**
- 快速上手
- 免费额度够用
- 专注于前端开发

**任务3：准备数据库（2天）**

**安装MySQL：**
- Windows: 下载 MySQL Installer
- macOS: `brew install mysql@8.0`
- 安装后启动服务

**创建数据库：**
```sql
-- 登录MySQL
mysql -u root -p

-- 创建数据库
CREATE DATABASE wechat_shopping CHARACTER SET utf8mb4;

-- 使用数据库
USE wechat_shopping;

-- 退出
exit;
```

**参考文档：** `docs/02-数据库设计.md` 中的完整表结构。

```
pages/home/
```

**在 .js 中定义数据：**
```javascript
Page({
  data: {
    message: 'Hello'
  }
});
```

**在 .wxml 中显示数据：**
```html
<view>{{message}}</view>
```

### 学习3：事件处理

**在 .wxml 中绑定事件：**
```html
<button bindtap="onTap">点击我</button>
```

**在 .js 中处理事件：**
```javascript
Page({
  onTap() {
    console.log('按钮被点击了');
```

## ⚠️ 重要提示

### 关于数据来源

**示例项目使用的是Mock数据：**

```javascript
// model/goods-list.js
const goodsList = [...];  // 假数据
```

**真实项目中需要：**

1. **方案A：Mock数据（个人账号推荐）**
   - 继续使用示例的数据格式
   - 只修改数据内容
   - 不连接真实服务器

2. **方案B：真实后端**
   - 需要开发后端接口
   - 前端调用接口获取数据
   - 需要配置服务器域名

**我的建议：先用方案A**
- 个人账号不支持支付
- 可以完整展示商品
- 可以模拟订单流程
- 适合展示和测试

### 关于支付功能

**个人账号的限制：**

| 功能 | 个人账号 | 企业账号 |
|------|---------|---------|
| 商品展示 | ✅ | ✅ |
| 购物车 | ✅ | ✅ |
| 订单创建 | ✅ | ✅ |
| 微信支付 | ❌ | ✅ |

**个人账号的解决方案：**
```javascript
// 创建订单时跳过支付
createOrder() {
  wx.showModal({
    title: '提示',
    content: '个人账号不支持在线支付，是否直接创建订单？',
    success: (res) => {
      if (res.confirm) {
        // 直接创建订单，标记为未支付
        this.createOrderWithoutPayment();
      }
    }
  });
}
```
