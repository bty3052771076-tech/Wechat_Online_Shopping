## 2. 环境准备

### 2.1 开发环境配置

#### 2.1.1 微信开发者工具安装

**下载地址**: https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

**安装步骤**:
1. 下载对应操作系统版本的开发者工具
2. 安装并启动工具
3. 使用微信扫码登录
4. 创建新项目或导入现有项目

**项目配置** (`project.config.json`):
```json
{
  "appid": "your_appid",
  "projectname": "在线购物系统",
  "compileType": "miniprogram",
  "setting": {
    "urlCheck": true,
    "es6": true,
    "postcss": true,
    "minified": true
  },
  "scripts": {
    "build": "npm run build"
  }
}
```

#### 2.1.2 Node.js 环境配置

**推荐版本**: Node.js 16.x LTS 或 18.x LTS

**安装步骤**:
```bash
# 使用 nvm 安装（推荐）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
source ~/.bashrc
nvm install 18
nvm use 18
nvm alias default 18

# 验证安装
node -v
npm -v
```

**npm 镜像配置**:
```bash
# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com

# 验证
npm config get registry
```

#### 2.1.3 MySQL 安装配置

**Windows 安装**:
1. 下载 MySQL Installer: https://dev.mysql.com/downloads/installer/
2. 运行安装程序，选择 MySQL Server
3. 设置 root 密码
4. 配置端口（默认3306）

**Linux 安装**:
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install mysql-server -y
sudo mysql_secure_installation

# CentOS/RHEL
sudo yum install mysql-server -y
sudo systemctl start mysqld
sudo systemctl enable mysqld
sudo mysql_secure_installation
```

**创建数据库和用户**:
```sql
-- 登录MySQL
mysql -u root -p

-- 创建数据库
CREATE DATABASE wechat_shop CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 创建用户并授权
CREATE USER 'shop_user'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON wechat_shop.* TO 'shop_user'@'localhost';
FLUSH PRIVILEGES;

-- 验证
SHOW DATABASES;
SELECT User, Host FROM mysql.user;
```

#### 2.1.4 Redis 安装配置

**Windows 安装**:
1. 下载 Redis for Windows: https://github.com/microsoftarchive/redis/releases
2. 解压并运行 redis-server.exe

**Linux 安装**:
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install redis-server -y

# CentOS/RHEL
sudo yum install redis -y

# 启动服务
sudo systemctl start redis
sudo systemctl enable redis

# 测试连接
redis-cli ping
```

**Redis 配置** (`/etc/redis/redis.conf`):
```conf
# 绑定地址
bind 127.0.0.1

# 端口
port 6379

# 设置密码

# 持久化配置

# 最大内存

```

### 2.2 生产环境准备

#### 2.2.1 云服务器购买与配置

**推荐云服务商**:
- 腾讯云：与微信生态集成较好
- 阿里云：生态完善
- 华为云：企业级服务

**服务器选择要点**:
1. **地域选择**: 选择距离用户最近的地域
2. **网络类型**: 专有网络（VPC）
3. **带宽**: 建议5Mbps以上，根据业务量调整
4. **系统盘**: SSD云硬盘，40GB以上
5. **数据盘**: 单独购买用于数据存储

**服务器初始化**:
```bash
# 更新系统
sudo apt update && sudo apt upgrade -y  # Ubuntu
sudo yum update -y                       # CentOS

# 安装基础工具
sudo apt install -y curl wget git vim net-tools htop iotop

# 配置时区
sudo timedatectl set-timezone Asia/Shanghai

# 配置SSH（修改端口、禁用root登录）
sudo vim /etc/ssh/sshd_config
# Port 2222
# PermitRootLogin no
# PasswordAuthentication no

# 重启SSH
sudo systemctl restart sshd
```

#### 2.2.2 域名购买与备案

**域名购买**:
- 推荐注册商：阿里云、腾讯云、GoDaddy
- 选择简短易记的域名
- 建议使用 .com 域名

**域名备案**（国内服务器必须）:
1. 登录云服务商控制台
2. 进入备案系统
3. 填写主体信息和网站信息
4. 上传证件照
5. 提交管局审核（通常7-20个工作日）

**域名规划示例**:
```
主域名: example.com

子域名规划:
- api.example.com          # API接口
- admin.example.com        # 管理后台
- img.example.com          # 静态资源
- m.example.com            # 移动端H5
```

#### 2.2.3 SSL证书申请与配置

**免费证书方案**:
1. **Let's Encrypt**（有效期90天，需自动续期）
2. **云服务商免费证书**（阿里云、腾讯云提供1年免费证书）

**使用 Let's Encrypt 申请证书**:
```bash
# 安装 certbot
sudo apt install certbot python3-certbot-nginx -y

# 申请证书
sudo certbot --nginx -d api.example.com -d admin.example.com

# 自动续期（certbot会自动添加cron任务）
sudo certbot renew --dry-run
```

**证书配置**:
```nginx
# Nginx SSL配置
server {
    listen 443 ssl http2;
    server_name api.example.com;

    ssl_certificate /etc/letsencrypt/live/api.example.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/api.example.com/privkey.pem;

    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;
    ssl_prefer_server_ciphers on;

    # ... 其他配置
}
```

#### 2.2.4 CDN配置

**CDN加速配置**:
1. 登录云服务商CDN控制台

### 3.3 数据库备份策略

#### 3.3.1 MySQL备份脚本

**全量备份脚本** (`/opt/scripts/mysql_backup.sh`):
```bash
#!/bin/bash

# 配置变量
BACKUP_DIR="/data/backups/mysql"
MYSQL_USER="shop_backup"
MYSQL_PASS="backup_password"
DATABASE="wechat_shop"
DATE=$(date +%Y%m%d_%H%M%S)
RETENTION_DAYS=7

# 创建备份目录
mkdir -p ${BACKUP_DIR}

# 执行备份
mysqldump -u${MYSQL_USER} -p${MYSQL_PASS} \
    --single-transaction \
    --routines \
    --triggers \
    --events \
    ${DATABASE} | gzip > ${BACKUP_DIR}/${DATABASE}_${DATE}.sql.gz

# 删除旧备份
find ${BACKUP_DIR} -name "*.sql.gz" -mtime +${RETENTION_DAYS} -delete

# 记录日志
echo "$(date '+%Y-%m-%d %H:%M:%S') Backup completed: ${DATABASE}_${DATE}.sql.gz" >> /var/log/mysql_backup.log
```

**定时任务配置**:
```bash
# 添加定时任务（每天凌晨2点执行）
crontab -e

# 添加以下行
0 2 * * * /opt/scripts/mysql_backup.sh
```

#### 3.3.2 Redis备份脚本

```bash
#!/bin/bash

# 配置变量
BACKUP_DIR="/data/backups/redis"
REDIS_PASS="your_password"
DATE=$(date +%Y%m%d_%H%M%S)

# 创建备份目录
mkdir -p ${BACKUP_DIR}

# 触发RDB持久化
redis-cli -a ${REDIS_PASS} BGSAVE

# 等待备份完成
sleep 10

# 复制备份文件
cp /var/lib/redis/dump.rdb ${BACKUP_DIR}/dump_${DATE}.rdb

# 压缩备份
gzip ${BACKUP_DIR}/dump_${DATE}.rdb
```

  env: cloud.DYNAMIC_CURRENT_ENV

const db = cloud.database();

// 云函数入口函数
exports.main = async (event, context) => {
  const { code } = event;

  try {
    // 获取用户信息
    const { result } = await cloud.openapi.auth.code2Session({
      jsCode: code
    });

    const { openid, unionid } = result;

    // 查询或创建用户
    const userRes = await db.collection('users').where({
      openid: openid
    }).get();

    let user;
    if (userRes.data.length === 0) {
      // 创建新用户
      const createRes = await db.collection('users').add({
        data: {
          openid,
          unionid,
          createTime: db.serverDate()
        }
      });
      user = { _id: createRes._id, openid, unionid };
    } else {
      user = userRes.data[0];
    }

    return {
      code: 0,
      data: user,
      message: 'success'
    };
  } catch (err) {
    console.error(err);
    return {
      code: -1,
      message: err.message
    };
  }
};

    await redis.expire(key, window);
  }

  if (current > limit) {
    throw new Error('请求过于频繁，请稍后再试');
  }

  return { remaining: limit - current };
};

// 使用示例

  const { openid } = ctx.state.user;

  // ...

// 加密函数
const encrypt = (text, key) => {
  const iv = crypto.randomBytes(16);
  const cipher = crypto.createCipheriv('aes-256-cbc', Buffer.from(key, 'hex'), iv);
  let encrypted = cipher.update(text);
  encrypted = Buffer.concat([encrypted, cipher.final()]);
  return iv.toString('hex') + ':' + encrypted.toString('hex');
};

// 解密函数
const decrypt = (text, key) => {
  const parts = text.split(':');
  const iv = Buffer.from(parts.shift(), 'hex');
  const encrypted = Buffer.from(parts.join(':'), 'hex');
  const decipher = crypto.createDecipheriv('aes-256-cbc', Buffer.from(key, 'hex'), iv);
  let decrypted = decipher.update(encrypted);
  decrypted = Buffer.concat([decrypted, decipher.final()]);
  return decrypted.toString();
};

#### 16.4.2 环境变量加密

**使用加密的环境变量**:
```javascript
const crypto = require('crypto');

// 加密环境变量
const encrypt = (text, key) => {
  const algorithm = 'aes-256-cbc';
  const iv = crypto.randomBytes(16);
  const cipher = crypto.createCipheriv(algorithm, Buffer.from(key, 'hex'), iv);
  let encrypted = cipher.update(text);
  encrypted = Buffer.concat([encrypted, cipher.final()]);
  return iv.toString('hex') + ':' + encrypted.toString('hex');
};

// 解密环境变量
const decrypt = (text, key) => {
  const algorithm = 'aes-256-cbc';
  const parts = text.split(':');
  const iv = Buffer.from(parts.shift(), 'hex');
  const encrypted = Buffer.from(parts.join(':'), 'hex');
  const decipher = crypto.createDecipheriv(algorithm, Buffer.from(key, 'hex'), iv);
  let decrypted = decipher.update(encrypted);
  decrypted = Buffer.concat([decrypted, decipher.final()]);
  return decrypted.toString();
};

// 使用
const encryptionKey = process.env.ENCRYPTION_KEY;
const dbPassword = decrypt(process.env.ENCRYPTED_DB_PASSWORD, encryptionKey);
```
