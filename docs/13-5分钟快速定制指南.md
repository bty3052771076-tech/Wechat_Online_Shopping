## ⚡ 快速开始

### 第1步：复制项目（1分钟）

**在文件管理器中：**

1. 找到 `E:\AI\cc+glm\example` 文件夹
2. 复制整个 `example` 文件夹
3. 粘贴到新位置，改名为 `Wechat_Online_Shopping`

```
E:\AI\cc+glm\
├── example/ (原始项目，不要修改)
└── Wechat_Online_Shopping/ (你的项目，在这里修改)
```

**或者直接在 example 中修改（如果你不需要保留原版）**

---

### 第2步：修改项目名称（1分钟）

**打开文件：** `project.config.json`

**修改：**
```json
{
  "projectname": "我的购物小程序"
}
```

**打开文件：** `app.json`

**修改：**
```json
{
  "window": {
    "navigationBarTitleText": "我的商城"
  }
}
```

**保存并查看：**
- 在开发者工具中点击「编译」
- 查看模拟器顶部的标题变化

---

### 第3步：修改商品数据（3分钟）

**打开文件：** `model/good.js`

**找到第11行左右，你会看到：**

```javascript
title: '白色短袖连衣裙荷叶边裙摆宽松韩版休闲纯白清爽优雅连衣裙',
```

**修改成：**

```javascript
title: '你的商品名称',
```

**继续往下看，找到价格（第19行左右）：**

```javascript
minSalePrice: 29800,
```

**这个价格单位是"分"，所以：**
- 29800 = 298元
- 9900 = 99元
- 19900 = 199元

**修改成你的价格：**

```javascript
minSalePrice: 9900,  // 改成99元
```

**保存并查看：**
- Ctrl+S 保存
- 点击「编译」
- 查看首页商品变化

## 📝 修改模板

为了让你更方便地修改商品数据，我创建了一个模板文件：

**复制以下代码，保存为 `model/goods-template.js`：**

```javascript
/**
 * 你的商品数据模板
 * 可以直接复制粘贴修改
 */

// 在这里定义你的商品
const yourGoods = [
  {
    id: '1',
    title: '商品1',
    price: 9900,  // 单位：分，9900 = 99元
    stock: 100,  // 库存数量
    image: '/assets/goods/product1.jpg',
    images: [
      '/assets/goods/product1.jpg',
      '/assets/goods/product1-2.jpg'
    ],
    category: '分类1',
    isNew: true,
    isHot: false
  },
  {
    id: '2',
    title: '商品2',
    price: 19900,  // 199元
    stock: 50,
    image: '/assets/goods/product2.jpg',
    images: [
      '/assets/goods/product2.jpg'
    ],
    category: '分类2',
    isNew: false,
    isHot: true
  }
  // ... 添加更多商品
];

// 导出你的商品列表
export function getYourGoodsList() {
  return yourGoods;
}
```

**使用方法：**

1. 复制上面的代码
2. 在 `model/` 目录创建新文件 `goods-template.js`
3. 粘贴并修改商品数据
4. 在需要的地方导入使用

## 🎨 颜色定制速查

**打开：** `app.json`

**找到：**
```json
{
  "tabBar": {
```

```json
{
```

## 🔧 常见问题快速解决

### 问题1：修改后没反应？

**解决：**
1. 确保按了 Ctrl+S 保存
2. 点击「编译」按钮
3. 或点击「清缓存」→「清除文件缓存」

### 问题2：图片不显示？

**解决：**
1. 确保图片路径正确
2. 小程序本地图片需要放在项目根目录
3. 或使用在线图片URL

### 问题3：报错怎么办？

**解决：**
1. 查看调试器的 Console 面板
2. 看错误提示的文件和行号
3. 检查是否有语法错误（少逗号、括号不匹配等）
