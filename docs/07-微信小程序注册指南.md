## 1. 注册前准备

### 1.1 为什么必须选择企业类型？

❌ **个人类型小程序的限制：**
- 不支持微信支付功能
- 无法开通微信商户号
- 部分接口权限受限
- 无法获取用户手机号
- 无法使用模板消息

✅ **企业类型小程序的优势：**
- 支持微信支付
- 可开通微信商户号
- 完整的接口权限
- 获取用户手机号
- 使用模板消息和订阅消息
- 更高的API调用限额

**结论：做电商小程序，必须选择企业类型！**

### 1.2 需要准备的材料

#### 企业账号注册必需材料：

| 材料名称 | 要求 | 说明 |
|---------|------|------|
| **营业执照** | 清晰照片或扫描件 | 必须在有效期内 |
| **法人身份证** | 正反面照片 | 必须清晰完整 |
| **对公账户** | 银行开户许可证或法人银行卡 | 用于验证企业身份 |
| **联系手机号** | 未注册过微信生态的手机号 | 用于接收验证码 |
| **联系邮箱** | 未注册过微信生态的邮箱 | 用于登录和通知 |
| **管理员身份证** | 正反面照片 | 小程序管理员 |
| **管理员手机号** | 实名认证的手机号 | 用于管理小程序 |
| **运营者身份证** | 正反面照片（可选） | 可由法人兼任 |

⚠️ **重要提示：**
- 所有照片必须是清晰的原件照片
- 不能是复印件、截图
- 照片格式：jpg、png
- 照片大小：不超过 2MB

#### 个人账号注册材料（不推荐）：

- 个人微信号
- 个人身份证
- 手机号
- 邮箱

## 3. 详细注册步骤

### 步骤1：访问微信公众平台

1. 打开浏览器，访问：**https://mp.weixin.qq.com/**
2. 点击右上角「**立即注册**」
3. 选择「**小程序**」

### 步骤2：填写账号信息

#### 3.1 填写邮箱

```
邮箱：your_email@example.com（请使用未注册过微信生态的邮箱）
密码：YourPassword123（请记住！）
确认密码：YourPassword123
```

⚠️ **注意：**
- 邮箱不能是已注册过公众号、服务号、订阅号的邮箱
- 建议使用企业邮箱
- 密码必须包含字母和数字

#### 3.2 验证邮箱

1. 点击「**注册**」后，系统会发送验证邮件
2. 登录邮箱，找到来自微信团队的邮件
3. 点击邮件中的验证链接
4. 返回注册页面继续

#### 3.3 选择主体类型

在主体类型页面，选择：

```
✅ 企业
  - 企业（需营业执照）
  - 政府
  - 媒体
  - 其他组织
❌ 个人（不支持支付）
```

**点击「企业」继续**

## 4. 企业认证流程

### 步骤3：企业信息登记

#### 4.1 填写企业基本信息

```
企业名称：北京XX科技有限公司（必须与营业执照一致）
统一社会信用代码：91110000XXXXXXXXXX（18位代码）
营业执照上传：点击上传清晰照片

经办人信息：
  姓名：张三
  身份证号：110XXXXXXXXXXXXXXX
  手机号：138XXXXXXXX
  验证码：输入短信验证码
```

#### 4.2 上传营业执照照片

要求：
- ✅ 营业执照原件照片
- ✅ 照片清晰，文字可辨认
- ✅ 完整显示营业执照所有信息
- ❌ 不要复印件、截图
- ❌ 不要有遮挡、反光
- ❌ 不要 PS 处理过

#### 4.3 填写对公账户信息

**方式一：使用开户许可证（推荐）**

```
开户许可证：
  - 上传开户许可证照片
  - 账户名称：北京XX科技有限公司
  - 账号：XXXXXXXXXXXXXXX
  - 开户行：中国工商银行北京XX支行
```

**方式二：使用法人银行卡**

```
法人银行卡：
  - 法人姓名：张三
  - 法人身份证号：110XXXXXXXXXXXXXXX
  - 银行卡号：XXXXXXXXXXXXXXX
  - 开户行：中国工商银行
```

⚠️ **重要：**
- 对公账户信息必须与营业执照一致
- 用于验证企业真实性
- 用于微信支付结算

#### 4.4 提交审核

填写完所有信息后：
1. 检查信息是否准确
2. 勾选「我同意并遵守……」
3. 点击「**提交审核**」
4. **支付认证费用：300元/年**

### 步骤4：支付认证费用

#### 支付方式

1. **微信支付**（推荐）
   - 扫码支付 300 元
   - 即时到账

2. **银行转账**
   - 按照提示转账
   - 到账时间：1-3个工作日

⚠️ **注意：**
- 300元是年度认证费，每年都需要续费
- 认证有效期为1年
- 到期前会提醒续费

### 步骤5：等待审核

#### 审核时间

- **正常审核**：1-3个工作日
- **高峰期**：可能延长至 7 个工作日

#### 审核进度查询

1. 登录微信公众平台
2. 查看「设置」→「基本配置」
3. 状态显示：
   - 「审核中」：正在审核
   - 「审核通过」：可以使用
   - 「审核驳回」：需要修改后重新提交

#### 审核结果通知

- 短信通知
- 邮件通知
- 站内消息

如果审核被驳回：
1. 查看驳回原因
2. 修改相关材料
3. 重新提交审核

## 5. 获取开发密钥

### 步骤6：登录小程序后台

审核通过后，使用注册邮箱登录：

1. 访问：https://mp.weixin.qq.com/
2. 输入邮箱和密码
3. 输入验证码（手机或邮箱）
4. 登录成功

### 步骤7：获取 AppID

#### 7.1 查看 AppID

1. 点击左侧菜单「**设置**」
2. 点击「**基本配置**」
3. 找到「**开发者ID**」部分

你会看到：

```
小程序ID (AppID): wxXXXXXXXXXXXXXXXX
小程序密钥 (AppSecret): XXXXXXXXXXXXXXXXXXXX
```

**重要说明：**
- **AppID**：小程序的唯一标识符
- **AppSecret**：用于调用微信接口的密钥

#### 7.2 保存 AppID 和 AppSecret

⚠️ **非常重要：**
- **截图保存** AppID 和 AppSecret
- **不要分享**给他人
- **不要提交**到公开代码仓库
- 后续开发会用到

**配置到项目中：**

```javascript
// config/config.js
module.exports = {
  wechat: {
    appId: 'wxXXXXXXXXXXXXXXXX',  // 你的 AppID
    appSecret: 'XXXXXXXXXXXXXXXXXX'  // 你的 AppSecret
  }
};
```

## 6. 微信支付申请

### 步骤8：申请微信支付商户号

⭐ **关键：电商小程序必须开通微信支付！**

#### 8.1 访问微信支付商户平台

1. 在小程序后台，点击「**微信支付**」
2. 或直接访问：https://pay.weixin.qq.com/
3. 点击「**立即接入**」

#### 8.2 选择接入方式

选择「**小程序接入**」

#### 8.3 填写商户信息

```
1. 经营主体：
   ✅ 企业（必须与小程序主体一致）

2. 经营信息：
   - 行业类目：电子商务（选择最接近的）
   - 经营场景：小程序商城
   - 销售商品描述：售卖XX类商品

3. 结算账户：
   - 经营者姓名：张三
   - 结算银行卡号：XXXXXXXXXXXXXXX
   - 开户行：中国工商银行XX支行
   - 结算身份信息：企业法人/经营者
```

#### 8.4 上传材料

需要上传：
1. 营业执照
2. 法人身份证
3. 组织机构代码证（如有）
4. 门头照片（店铺照片）

#### 8.5 提交审核

1. 确认信息无误
2. 点击「**提交申请**」
3. 等待审核（1-7个工作日）

#### 8.6 签约协议

审核通过后：
1. 登录微信支付商户平台
2. 签署《微信支付服务协议》
3. 完成！

#### 8.7 配置支付密钥

##### 8.7.1 设置 API 密钥

1. 登录微信支付商户平台
2. 进入「**账户中心**」→「**API安全**」
3. 点击「**设置API密钥**」
4. 设置一个32位的密钥（例如：your_32_character_api_key_here）
5. **保存密钥**（后续开发会用）

**密钥要求：**
- 长度：32个字符
- 字符：数字、字母（区分大小写）
- 不要使用特殊符号

##### 8.7.2 下载商户证书

1. 在「**API安全**」页面
2. 点击「**下载证书**」
3. 安装证书工具
4. 生成并下载证书
5. 解压得到：
   - `apiclient_cert.pem`（商户证书）
   - `apiclient_key.pem`（商户密钥）

**安全提示：**
- 证书文件必须妥善保管
- 不要上传到公开代码仓库
- 建议存放在项目根目录的 `certs/` 文件夹

### 步骤9：关联小程序和商户号

1. 登录微信支付商户平台
2. 进入「**产品中心**」→「**开发配置**」
3. 点击「**关联AppID**」
4. 输入你的小程序 AppID
5. 确认关联

## 7. 常见问题

### Q1: 注册时提示「邮箱已被占用」？

**原因：** 该邮箱已注册过微信生态账号（公众号、服务号等）

**解决方法：**
- 使用其他邮箱注册
- 或注销旧账号后重新注册

### Q2: 企业认证被驳回怎么办？

**常见驳回原因：**
1. 营业执照不清晰
2. 营业执照已过期
3. 企业名称与营业执照不符
4. 对公账户信息错误
5. 联系方式无法联系

**解决方法：**
1. 查看驳回原因
2. 修改相关材料
3. 重新提交审核

### Q3: 认证费用已支付但未到账？

**解决方法：**
- 微信支付：通常即时到账
- 银行转账：等待1-3个工作日
- 如超时未到账，联系微信客服

### Q4: AppSecret 忘记了怎么办？

**解决方法：**
1. 登录小程序后台
2. 进入「设置」→「基本配置」
3. 点击「**重置**」AppSecret
4. 通过验证后重新生成
5. **旧密钥立即失效**

⚠️ **注意：重置后需要更新项目配置！**

### Q5: 个人账号可以升级为企业账号吗？

**答案：** 不可以

**原因：** 账号主体类型注册后不能更改

**建议：**
- 如果做电商，直接注册企业账号
- 如果已注册个人账号，需要重新注册企业账号

### Q6: 微信支付审核被驳回？

**常见原因：**
1. 营业范围不符
2. 商品描述不清晰
3. 门头照片不合格
4. 结算账户信息错误

**解决方法：**
1. 仔细阅读驳回原因
2. 修改相关材料
3. 重新提交

### Q7: 如何查询审核进度？

**方法一：**
- 登录小程序后台
- 查看「设置」→「基本配置」

**方法二：**
- 登录微信支付商户平台
- 查看「消息中心」

**方法三：**
- 等待短信或邮件通知

## 8. 后续配置

### 步骤10：配置小程序信息

#### 8.1 设置小程序名称

1. 进入「**设置**」→「**基本设置**」
2. 点击「**名称**」旁边的「**修改**」
3. 输入小程序名称（4-30个字符）

**命名规范：**
- 不能包含「微信」、「腾讯」等词汇
- 不能使用品牌词（需授权）
- 不能使用特殊符号
- 建议使用：品牌名 + 行业词

例如：
- ✅ 优选商城
- ✅ XX便利店
- ❌ 微信商城（含「微信」）
- ❌ 京东商城（侵权）

#### 8.2 上传小程序头像

1. 点击「**头像**」旁边的「**修改**」
2. 上传 1:1 比例的图片
3. 推荐：**512 x 512 像素**
4. 格式：jpg、png
5. 大小：不超过 2MB

#### 8.3 填写小程序简介

1. 点击「**简介**」旁边的「**修改**」
2. 输入小程序功能介绍（4-120个字符）

例如：
```
「XX商城」是专业的在线购物平台，
提供商品浏览、在线支付、订单管理、
物流查询等完整的购物体验。
```

#### 8.4 设置服务类目

1. 点击「**服务类目」」
2. 选择「**电商 > 百货/生活用品**」
3. 或选择最接近的类目

**注意：**
- 类目必须与实际经营范围相符
- 部分类目需要额外资质

### 步骤11：配置开发设置

#### 8.5 配置服务器域名

1. 进入「**开发**」→「**开发管理**」→「**开发设置**」
2. 找到「**服务器域名**」
3. 点击「**修改**」
4. 填写以下域名：

```
request合法域名：
https://api.yourdomain.com

socket合法域名：
wss://api.yourdomain.com

uploadFile合法域名：
https://upload.yourdomain.com

downloadFile合法域名：
https://static.yourdomain.com
```

⚠️ **注意：**
- 域名必须是 HTTPS
- 域名必须已备案
- 开发阶段可勾选「不校验合法域名」
- 上线前必须配置正确的域名

#### 8.6 配置业务域名

```
业务域名：
www.yourdomain.com
yourdomain.com
```

**用途：** 小程序内打开网页

#### 8.7 配置用户隐私保护

1. 进入「**设置**」→「**基本设置**」
2. 找到「**用户隐私保护指引**」
3. 填写隐私政策
4. 说明收集了哪些用户信息、如何使用等
