## 2. 个人账号注册步骤

### 步骤1：访问注册页面

1. 打开浏览器，访问：**https://mp.weixin.qq.com/**
2. 点击右上角「**立即注册**」
3. 选择「**小程序**」

### 步骤2：填写账号信息

```
邮箱：your_email@example.com（任意未注册过微信的邮箱）
密码：YourPassword123（请记住）
确认密码：YourPassword123
```

点击「**注册**」

### 步骤3：验证邮箱

1. 登录邮箱，找到来自微信团队的邮件
2. 点击邮件中的验证链接
3. 跳转回注册页面

### 步骤4：选择主体类型

⚠️ **重要：选择「个人」**

```
请选择你的主体类型：

❌ 企业（需要营业执照、300元认证费）
❌ 政府
❌ 媒体
❌ 其他组织
✅ 个人 ← 选择这个！
```

**点击「个人」继续**

### 步骤5：填写个人信息

```
主体名称：张三（你的真实姓名）
身份证号：110XXXXXXXXXXXXXXX
身份证照片：上传身份证正反面照片

管理员信息：
  姓名：张三
  身份证号：110XXXXXXXXXXXXXXX
  手机号：138XXXXXXXX
  验证码：输入短信验证码
```

**上传身份证照片要求：**
- ✅ 清晰的原件照片（正反面都要）
- ✅ 四个角完整
- ✅ 信息清晰可见
- ❌ 不要复印件
- ❌ 不要截图
- ❌ 不要有反光

### 步骤6：提交审核

1. 勾选「我同意……」
2. 点击「**提交**」

**注意：个人账号是免费的，无需支付费用！**

### 步骤7：等待审核

- **审核时间**：通常 1-3 个工作日
- **审核结果**：通过短信和邮件通知
- **查询进度**：登录小程序后台查看

### 步骤8：获取 AppID

审核通过后：

1. 使用注册邮箱登录
2. 进入「**设置**」→「**基本配置**」
3. 复制保存：

```
小程序ID (AppID): wxXXXXXXXXXXXXXXXX
```

**保存到项目配置：**

```javascript
// config/config.js

  wechat: {
    appId: 'wxXXXXXXXXXXXXXXXX',  // 你的 AppID
```

## 4. 如何在个人账号下开发

### 4.1 功能调整建议

#### 订单流程调整

**原流程（企业版）：**
```
商品 → 加入购物车 → 结算 → 微信支付 → 订单确认 → 发货
```

**调整后（个人版）：**
```
商品 → 加入购物车 → 结算 → 模拟支付 → 订单确认 → 发货
```

### 4.2 模拟支付功能

#### 方案一：直接确认订单（推荐）

```javascript
// 订单确认页面
Page({
  // 提交订单（跳过支付）
  submitOrder() {
    wx.showModal({
      title: '确认订单',
      content: '个人账号暂不支持在线支付，是否直接创建订单？',
      success: (res) => {
        if (res.confirm) {
          this.createOrder();
        }
      }
    });
  },

  // 直接创建订单
  async createOrder() {
    wx.showLoading({ title: '创建中...' });

    try {
      const res = await services.order.create({
        // 订单数据
        items: this.data.goodsList,
        address: this.data.address,
        amount: this.data.totalAmount,
        paymentStatus: 'unpaid',  // 标记为未支付
        paymentMethod: 'mock'      // 模拟支付
      });

      wx.hideLoading();
      wx.showToast({ title: '订单创建成功', icon: 'success' });

      // 跳转到订单详情
      setTimeout(() => {
        wx.redirectTo({
          url: `/pages/order/order-detail?id=${res.data.orderId}`
        });
      }, 1500);

    } catch (error) {
      wx.hideLoading();
      wx.showToast({ title: '创建失败', icon: 'none' });
    }
  }
});
```

#### 方案二：模拟支付弹窗

#### 方案三：货到付款模式

```javascript
// 选择支付方式
selectPaymentMethod(e) {
  const method = e.currentTarget.dataset.method;

  if (method === 'wechat') {
    // 微信支付（个人账号不支持）
    wx.showModal({
      title: '提示',
      content: '个人账号暂不支持在线支付，请选择其他支付方式',
      showCancel: false
    });
    return;
  }

  if (method === 'cod') {
    // 货到付款
    this.setData({ paymentMethod: 'cod' });
  }
}

// 提交订单
async submitOrder() {
  const order = {
    items: this.data.goodsList,
    address: this.data.address,
    amount: this.data.totalAmount,
    paymentMethod: this.data.paymentMethod, // cod: 货到付款
    paymentStatus: 'unpaid'
  };

  await services.order.create(order);
  wx.showToast({ title: '订单创建成功', icon: 'success' });
}
```

## 6. 后续升级方案

### 6.1 什么时候需要升级？

**建议升级企业账号的情况：**

- ✅ 正式上线运营
- ✅ 需要真实支付功能
- ✅ 需要获取用户手机号
- ✅ 需要更多的API调用次数
- ✅ 商业化运营

### 6.2 如何升级？

**方案一：注册新的企业账号（推荐）**

```
1. 保持个人账号用于开发测试
2. 注册新的企业账号
3. 在企业账号中开发正式版本
4. 两套代码同时维护
```

**优势：**
- 个人账号可以继续用于测试
- 不会影响正在使用的用户
- 风险更低

**方案二：直接变更主体类型**

⚠️ **不支持！** 微信小程序的账号主体类型注册后不能更改。

**只能：**
1. 注销个人账号
2. 重新注册企业账号

**不推荐！因为：**
- 原有用户会丢失
- 数据需要迁移
- 注销需要30天

**建议：注册新的企业账号！**

### 6.3 代码迁移方案

**从个人版迁移到企业版，只需修改支付相关代码：**

```javascript
// 个人版
submitOrder() {
  // 模拟支付
  this.mockPayment();
}

// 企业版
submitOrder() {
  // 微信支付
  this.wechatPay();
}

// 兼容代码（推荐）
submitOrder() {
  const accountType = config.wechat.accountType; // 'personal' | 'enterprise'

  if (accountType === 'personal') {
    // 个人账号：模拟支付
    this.mockPayment();
  } else {
    // 企业账号：微信支付
    this.wechatPay();
  }
}
```

**配置文件：**

```javascript
// config/config.js
module.exports = {
  wechat: {
    accountType: 'personal',  // 'personal' | 'enterprise'
    appId: 'wxXXXXXXXXXXXXXXXX',
    // 个人账号不需要配置支付
    payment: {
      // 企业账号才需要
      merchantId: '',
      apiKey: ''
    }
  }
};
```
