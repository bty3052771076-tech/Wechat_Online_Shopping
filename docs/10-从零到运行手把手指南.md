## 📋 接下来的5个步骤

### 步骤1：启动微信开发者工具并登录（2分钟）

#### 1.1 打开工具

**Windows系统：**
- 双击桌面图标「微信web开发者工具」
- 或从开始菜单找到并打开

**Mac系统：**
- 在「启动台」找到「微信开发者工具」
- 或在「应用程序」文件夹中找到

#### 1.2 微信扫码登录

打开工具后，你会看到一个二维码：

```
┌─────────────────────┐
│                     │
│    [二维码图片]      │
│                     │
│  请使用微信扫码登录  │
│                     │
└─────────────────────┘
```

**操作步骤：**
1. 拿出手机，打开微信
2. 点击「发现」→「扫一扫」
3. 扫描电脑上的二维码
4. 手机上确认登录

**登录成功后：**
- 电脑显示工具主界面
- 右上角显示你的微信头像

---

### 步骤2：注册个人小程序账号（5-10分钟）

⚠️ **重要：还没有小程序账号？需要先注册！**

#### 2.1 访问注册页面

1. 在电脑浏览器打开：**https://mp.weixin.qq.com/**
2. 点击右上角「**立即注册**」
3. 选择「**小程序**」

#### 2.2 填写账号信息

```
邮箱：your_email@example.com
  （使用任意未注册过微信的邮箱）

密码：YourPassword123
  （记住这个密码！）

确认密码：YourPassword123
```

点击「**注册**」，然后去邮箱验证。

#### 2.3 选择主体类型

**重要选择：**
- ❌ 不要选「企业」（需要营业执照、300元）
- ✅ **选择「个人」**（免费、快速）

点击「**个人」」继续。

#### 2.4 填写个人信息

```
主体名称：张三（你的真实姓名）

身份证号：110XXXXXXXXXXXXXXX
（输入你的身份证号）

身份证照片：
- 点击上传身份证正面照片
- 点击上传身份证反面照片
（照片要清晰，四个角都要拍进去）

管理员信息：
  姓名：张三
  身份证号：110XXXXXXXXXXXXXXX
  手机号：138XXXXXXXX（你的手机号）
  验证码：（输入手机收到的验证码）
```

点击「**提交**」。

**个人账号是免费的，不需要支付！**

#### 2.5 等待审核

- **审核时间**：通常 1-3 个工作日
- **审核结果**：短信通知

⚠️ **等等！我可以先用测试号吗？**

**可以！** 如果不想等审核，可以先用测试号：

1. 在微信开发者工具中创建项目时
2. AppID 选择「**测试号**」
3. 可以立即开始开发
4. 但功能有限制，不能发布

**我的建议：**
- 现在先用测试号开始开发 ✅
- 同时注册个人账号（等审核）
- 审核通过后替换成真实 AppID

### 步骤3：创建第一个小程序项目（3分钟）

#### 3.1 打开项目窗口

微信开发者工具登录后，有两种方式：

**方式一：首次打开**
- 自动弹出「创建项目」窗口

**方式二：已打开过**
- 点击左上角「**+**」号
- 或点击「**项目**」→「**新建项目**」

#### 3.2 填写项目信息

你会看到这样的界面：

```
┌──────────────────────────────────────┐
│  小程序                                │
│  ┌────────────────────────────────┐  │
│  │ 项目名称：MyFirstMiniProgram    │  │
│  │                                 │  │
│  │ 目录：C:\Users\YourName\First... │  │
│  │                                 │  │
│  │ AppID：[测试号 ▼]               │  │
│  │                                 │  │
│  │ 开发模式：                       │  │
│  │  JavaScript ○ TypeScript        │  │
│  │                                 │  │
│  │ 后端服务：                       │  │
│  │  不使用云服务 ○ 使用云开发      │  │
│  │                                 │  │
│  │ 模板：JavaScript - 基础模板     │  │
│  └────────────────────────────────┘  │
│                                        │
│  [新建]  [取消]                        │
└──────────────────────────────────────┘
```

**具体填写：**

**1. 项目名称：**
```
MyFirstMiniProgram
（或任意你喜欢的名字）
```

**2. 目录：**
```
选择一个空文件夹
例如：C:\Users\YourName\Documents\MyMiniProgram
```
- 点击「**选择**」按钮
- 新建一个空文件夹
- 选择这个文件夹

**3. AppID：**
```
点击下拉框 → 选择「测试号」
```
**为什么选测试号？**
- ✅ 不需要注册
- ✅ 立即可用
- ✅ 可以学习开发
- ❌ 不能发布上线

**4. 开发模式：**
```
选择「JavaScript」
（推荐初学者使用）
```

**5. 后端服务：**
```
选择「不使用云服务」
```

**6. 模板：**
```
选择「JavaScript - 基础模板」
```

#### 3.3 点击「新建」

点击「**新建**」按钮后，工具会：
1. 创建项目文件
2. 打开项目界面
3. 自动编译代码

### 步骤4：认识开发工具界面（2分钟）

#### 4.1 主界面布局

打开项目后，你会看到：

```
┌─────────────────────────────────────────────────┐
│  菜单栏：项目 文件 工具 面板 设置                   │
├──────────────┬───────────────────────────────────┤
│              │  工具栏：[模拟器] [调试器] [清缓存]   │
│   文件列表    ├───────────────────────────────────┤
│              │                                   │
│  ├─ pages/  │         模拟器                    │
│  │  └─ index/│    (显示小程序效果)              │
│  ├─ utils/  │                                   │
│  │  └── ...  │    【Hello World!】              │
│  │           │                                   │
│  ├─ app.js  │    [ 按钮 ]                       │
│  ├─ app.json│                                   │
│  └─ app.wxss│                                   │
│              │                                   │
│  代码编辑区  ├───────────────────────────────────┤
│              │         调试器                     │
│  （显示代码） │  [Console][Storage][Network]       │
│              │                                   │
└──────────────┴───────────────────────────────────┘
```

#### 4.2 各区域说明

| 区域 | 作用 | 说明 |
|------|------|------|
| **文件列表** | 显示项目文件 | 左侧，类似资源管理器 |
| **代码编辑区** | 编辑代码 | 中间，显示当前文件内容 |
| **模拟器** | 预览效果 | 右上，显示小程序在手机上的样子 |
| **调试器** | 调试代码 | 右下，查看日志、错误等 |

#### 4.3 尝试修改代码

**让我们来修改第一个小程序：**

1. 在文件列表中找到 `pages/index/index.wxml`
2. 点击打开这个文件
3. 你会看到代码：

```html
<view class="container">
  <text>Hello World!</text>
  <button bindtap="onTap">点击我</button>
</view>
```

4. 把 `Hello World!` 改成 `你好，小程序！`：

```html
<view class="container">
  <text>你好，小程序！</text>
  <button bindtap="onTap">点击我</button>
</view>
```

5. **看模拟器**：文字会自动变成「你好，小程序！」

**恭喜你！你已经修改了你的第一个小程序！** 🎉

## ⚠️ 常见问题快速解决

### Q1: 找不到「导入项目」？

**解决：**
1. 关闭当前项目：「**项目**」→「**关闭项目**」
2. 看到大大的「**+**」号
3. 点击「**+**」→「**导入项目**」

### Q2: 导入后提示错误？

**错误：`module "miniprogram_npm/tdesign-miniprogram/components/xxx" is not defined`

**解决：**
1. 点击「**工具**」→「**构建 npm**」
2. 等待构建完成（可能需要几分钟）
3. 构建完成后重新编译

### Q3: 模拟器显示空白？

**解决：**
1. 检查 AppID 是否正确
2. 点击「**清缓存**」→「**全部清除**」
3. 点击「**编译**」按钮

### Q4: 代码修改后没反应？

**解决：**
1. 确保已保存文件（Ctrl+S）
2. 点击「**编译**」按钮
3. 或点击「**清缓存**」→「**清除文件缓存**」
