## 1. 硬件要求

### 最低配置
- **CPU**: 双核处理器
- **内存**: 8GB RAM
- **硬盘**: 20GB 可用空间
- **操作系统**: Windows 10/11, macOS 10.14+, 或 Linux

### 推荐配置
- **CPU**: 四核或更高
- **内存**: 16GB RAM 或更高
- **硬盘**: SSD, 50GB+ 可用空间
- **操作系统**: Windows 11, macOS, 或 Ubuntu 20.04+

## 3. 数据库环境

### 3.1 MySQL（必需）

#### Windows 安装

**方式一：使用 MySQL Installer（推荐）**

1. 访问 [https://dev.mysql.com/downloads/installer/](https://dev.mysql.com/downloads/installer/)
2. 下载 MySQL Installer for Windows
3. 运行安装程序，选择自定义安装
4. 选择以下组件：
   - MySQL Server 8.0.x
   - MySQL Workbench
   - MySQL Shell
5. 设置 root 密码（请记住！）
6. 启动 MySQL 服务

**方式二：使用 Docker（推荐开发者）**

```bash
# 拉取 MySQL 镜像
docker pull mysql:8.0

# 运行 MySQL 容器
docker run --name mysql \
  -e MYSQL_ROOT_PASSWORD=your_password \
  -p 3306:3306 \
  -v mysql_data:/var/lib/mysql \
  -d mysql:8.0
```

#### macOS 安装

```bash
# 使用 Homebrew
brew install mysql@8.0

# 启动 MySQL 服务
brew services start mysql@8.0

# 设置 root 密码
mysql_secure_installation
```

#### 验证安装

```bash
# Windows
mysql -u root -p

# macOS / Linux
mysql -u root -p

# 输入密码后，看到以下提示表示成功
Welcome to the MySQL monitor...
mysql>
```

#### 创建项目数据库

```sql
-- 登录 MySQL
mysql -u root -p

-- 创建数据库
CREATE DATABASE wechat_shopping CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 创建用户（可选，用于开发）
CREATE USER 'wechat_user'@'localhost' IDENTIFIED BY 'your_password';

-- 授权
GRANT ALL PRIVILEGES ON wechat_shopping.* TO 'wechat_user'@'localhost';

-- 刷新权限
FLUSH PRIVILEGES;

-- 退出
exit;
```

### 3.2 Redis（必需）

#### Windows 安装

**方式一：使用 Memurai（Redis for Windows）**

1. 访问 [https://www.memurai.com/get-memurai-for-windows](https://www.memurai.com/get-memurai-for-windows)
2. 下载并安装 Memurai（Redis 兼容）
3. 启动服务

**方式二：使用 WSL2 + Redis（推荐）**

```bash
# 在 WSL2 Ubuntu 中安装 Redis
sudo apt update
sudo apt install redis-server

# 启动 Redis
sudo service redis-server start

# 验证
redis-cli ping
# 应返回: PONG
```

**方式三：使用 Docker（推荐）**

```bash
# 拉取 Redis 镜像
docker pull redis:7-alpine

# 运行 Redis 容器
docker run --name redis \
  -p 6379:6379 \
  -d redis:7-alpine
```

#### macOS 安装

```bash
# 使用 Homebrew
brew install redis

# 启动 Redis
brew services start redis

# 验证
redis-cli ping
# 应返回: PONG
```

#### 验证 Redis

```bash
# 启动 Redis CLI
redis-cli

# 测试
127.0.0.1:6379> ping
PONG

127.0.0.1:6379> set test "hello"
OK

127.0.0.1:6379> get test
"hello"
```

## 5. 云服务准备

### 5.1 微信小程序账号

#### 注册流程

1. **访问微信公众平台**
   - 网址: [https://mp.weixin.qq.com](https://mp.weixin.qq.com)
   - 点击"立即注册"
   - 选择"小程序"

2. **填写账号信息**
   - 邮箱（未注册过微信生态的邮箱）
   - 密码
   - 确认邮箱

3. **选择主体类型**
   - ❌ 个人（不支持支付功能）
   - ✅ **企业**（推荐，必须选择企业才能开通支付）
   - ✅ 政府
   - ✅ 媒体
   - ✅ 其他组织

4. **企业认证**（必须）
   - 上传营业执照
   - 填写企业信息
   - 法人身份证信息
   - 对公账户信息
   - **认证费**: 300元/年

5. **完善小程序信息**
   - 小程序名称
   - 小程序简介
   - 小程序头像（1:1）
   - 服务类目

#### 获取 AppID

登录小程序后台：
- 开发 -> 开发设置 -> 开发者ID
- **AppID (小程序ID)**: wxXXXXXXXXXXXXXXXX
- **AppSecret (小程序密钥)**: 点击"重置"获取，请妥善保管

### 5.2 微信支付商户号

#### 申请流程

1. **访问微信支付商户平台**
   - 网址: [https://pay.weixin.qq.com](https://pay.weixin.qq.com)
   - 使用小程序账号登录

2. **提交申请资料**
   - 营业执照
   - 法人身份证
   - 开户许可证或法人银行卡
   - 门头照片（一张）

3. **等待审核**
   - 审核时间: 1-7个工作日
   - 审核通过后，商户号会关联到小程序

4. **配置 API**
   - 登录商户平台
   - 账户中心 -> API安全 -> 设置API密钥
   - 下载证书（用于退款等操作）

### 5.3 云服务账号（选择一个）

#### 腾讯云（推荐）

1. **注册账号**
   - 网址: [https://cloud.tencent.com](https://cloud.tencent.com)
   - 完成实名认证

2. **选择服务**

**方案一：云开发（推荐初创）**

- 访问: [https://console.cloud.tencent.com/tcb](https://console.cloud.tencent.com/tcb)
- 创建环境
- 选择基础版（19.9元/月）

**方案二：云服务器（传统部署）**

- 访问: [https://console.cloud.tencent.com/cvm](https://console.cloud.tencent.com/cvm)
- 购买云服务器
  - 地区: 选择离用户最近的区域
  - 配置: 2核4G起步
  - 镜像: CentOS 7.x 或 Ubuntu 20.04
  - 带宽: 5Mbps起步

3. **开通云存储 COS**
   - 网址: [https://console.cloud.tencent.com/cos](https://console.cloud.tencent.com/cos)
   - 创建存储桶
   - 设置权限为: 公有读私有写

4. **开通短信服务**
   - 网址: [https://console.cloud.tencent.com/sms](https://console.cloud.tencent.com/sms)
   - 创建签名和模板
   - 等待审核（1-3个工作日）

#### 阿里云（备选）

1. **注册账号**
   - 网址: [https://aliyun.com](https://aliyun.com)
   - 完成实名认证

2. **开通云服务器 ECS**
   - 网址: [https://ecs.console.aliyun.com](https://ecs.console.aliyun.com)

   - 网址: [https://dysms.console.aliyun.com](https://dysms.console.aliyun.com)

#### 购买域名

1. **推荐服务商**

   - 新增备案: 10-15个工作日
   - 建议尽早开始！

3. **备案所需材料**
   - 个人: 身份证、核验照片
   - 企业: 营业执照、法人身份证、核验照片

## 6. 环境变量配置

### 6.1 创建配置文件

在项目根目录创建配置文件：

```bash
# 进入项目目录
cd Wechat_Online_Shopping

# 创建配置文件
touch config/config.development.js
touch config/config.production.js
```

### 6.2 配置文件示例

#### `config/config.js`（主配置）

```javascript
const env = process.env.NODE_ENV || 'development';

module.exports = {
  // 当前环境
  env,

  // API 配置
  api: {
    baseUrl: env === 'production'
      ? 'https://api.yourdomain.com'
      : 'http://localhost:3000',
    timeout: 30000
  },

  // 数据库配置
  database: {
    host: env === 'production' ? 'your-production-host' : 'localhost',
    port: 3306,
    username: env === 'production' ? 'prod_user' : 'root',
    password: env === 'production' ? 'prod_password' : 'your_password',
    database: 'wechat_shopping',
    charset: 'utf8mb4'
  },

  // Redis 配置
  redis: {
    host: env === 'production' ? 'your-redis-host' : 'localhost',
    port: 6379,
    password: env === 'production' ? 'your-redis-password' : ''
  },

  // 微信小程序配置
  wechat: {
    appId: 'your_wechat_appid',
    appSecret: 'your_wechat_appsecret'
  },

  // 微信支付配置
  payment: {
    merchantId: 'your_merchant_id',
    apiKey: 'your_api_key',
    certPath: '/path/to/cert/apiclient_cert.pem',
    keyPath: '/path/to/cert/apiclient_key.pem'
  },

  // 云存储配置
  storage: {
    provider: 'tencent', // 'tencent' | 'aliyun'
    region: 'ap-guangzhou',
    bucket: 'your-bucket-name',
    secretId: 'your_secret_id',
    secretKey: 'your_secret_key'
  },

  // 短信配置
  sms: {
    provider: 'tencent', // 'tencent' | 'aliyun'
    appId: 'your_sms_appid',
    appKey: 'your_sms_appkey',
    signName: 'your_sign_name',
    templateId: 'your_template_id'
  }
};
```

### 6.3 环境变量文件

创建 `.env` 文件（不要提交到 Git）：

```bash
# .env
NODE_ENV=development

# 数据库
DB_HOST=localhost
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=your_password
DB_DATABASE=wechat_shopping

# Redis
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=

# 微信
WECHAT_APPID=your_wechat_appid
WECHAT_APPSECRET=your_wechat_appsecret

# 微信支付
PAYMENT_MERCHANT_ID=your_merchant_id
PAYMENT_API_KEY=your_api_key

# 云存储
STORAGE_SECRET_ID=your_secret_id
STORAGE_SECRET_KEY=your_secret_key

# 短信
SMS_APP_ID=your_sms_appid
SMS_APP_KEY=your_sms_appkey
```

**重要**: 将 `.env` 添加到 `.gitignore`：

```bash
# .gitignore
.env
config/config.*.js
!config/config.example.js
```

### 7.2 创建测试脚本

创建 `scripts/check-env.js`：

```javascript
const mysql = require('mysql2/promise');
const redis = require('redis');

async function checkDatabase() {
  console.log('检查 MySQL...');
  try {
    const connection = await mysql.createConnection({
      host: 'localhost',
      user: 'root',
      password: 'your_password'
    });
    console.log('✅ MySQL 连接成功');
    await connection.end();
  } catch (error) {
    console.log('❌ MySQL 连接失败:', error.message);
  }
}

async function checkRedis() {
  console.log('检查 Redis...');
  try {
    const client = redis.createClient();
    await client.connect();
    await client.ping();
    console.log('✅ Redis 连接成功');
    await client.quit();
  } catch (error) {
    console.log('❌ Redis 连接失败:', error.message);
  }
}

async function main() {
  await checkDatabase();
  await checkRedis();
  console.log('\n环境检查完成！');
}

main();
```

运行检查：

```bash
node scripts/check-env.js
```

## 8. 常见问题

### Q1: Node.js 安装后无法使用？
```bash
# Windows: 重新打开终端
# macOS: 添加到 PATH
echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

### Q2: MySQL 启动失败？
```bash
# Windows: 检查服务
# Win + R -> services.msc -> 找到 MySQL80 -> 启动

# macOS:
brew services start mysql@8.0

# Linux:
sudo systemctl start mysql
```

### Q3: Redis 连接不上？
```bash
# 检查 Redis 是否运行
redis-cli ping

# 如果没有运行，启动 Redis
# macOS:
brew services start redis

# Linux:
sudo service redis-server start

# Windows (WSL2):
sudo service redis-server start
```

### Q4: npm install 很慢？
```bash
# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com

# 或使用 cnpm
npm install -g cnpm --registry=https://registry.npmmirror.com
cnpm install
```

### Q5: 微信开发者工具无法预览？
- 确保已在微信小程序后台配置域名白名单
- 开发阶段可勾选"不校验合法域名"
- 位置: 微信开发者工具 -> 右上角详情 -> 本地设置

### Q6: 小程序无法上传代码？
- 确保 AppID 配置正确
- 确保已完成企业认证
- 检查 project.config.json 中的 appid
